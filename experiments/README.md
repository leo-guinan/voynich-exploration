# Experiments

This directory contains experimental runs and their logs.

## Structure

Each experiment should have:
- A descriptive name
- A log file in `logs/`
- Clear documentation of what was tested
- Results and conclusions

## Naming Convention

Use descriptive names:
- `compression_baseline_20240101`
- `position_effects_line_start`
- `section_comparison_currier_ab`

## Logging

All experiments should log:
- Input parameters
- Output files generated
- Key findings
- Time taken
- Any errors or warnings

---

*Document experiments as they are run.*
